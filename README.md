# Matchbox
Matchbox is a fast and portable programming language that is memory-safe and has an expressive, easy-to-learn syntax.

## Supported Platforms
Matchbox can run on the following platforms:

- Windows

## Installation
Pre-built binaries can be found at https://www.matchbox-lang.org/downloads.

## Getting Started
First, create a file named main.mb and add the following code:

```
var x = 10
var y = 20
var result = x + y

print(result)
```

Next, compile and run the program:

```
matchbox main.mb
```
