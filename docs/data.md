# Data
| Unit   | Bits | Bytes | Hex   | Common Uses / Notes                 |
| ------ | ---- | ----- | ----- | ----------------------------------- |
| bit    | 1    | 0.125 | 0x1   | Boolean values, flags               |
| nibble | 4    | 0.5   | 0x4   | Half byte, hex digit                |
| byte   | 8    | 1     | 0x8   | Characters, small integers          |
| hword  | 16   | 2     | 0x10  | Half word, CPU instructions         |
| word   | 32   | 4     | 0x20  | CPU register size, memory alignment |
| dword  | 64   | 8     | 0x40  | Double word, 64-bit integers        |
| qword  | 128  | 16    | 0x80  | Quad word, SIMD vectors             |
| oword  | 256  | 32    | 0x100 | Octaword, AVX 256-bit registers     |
| yword  | 512  | 64    | 0x200 | Double quadword, AVX-512 registers  |
| zword  | 1024 | 128   | 0x400 | Very large block, cache lines       |
